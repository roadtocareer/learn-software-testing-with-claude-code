Skills in Claude Code

Skills are reusable slash-command modules that give Claude a specific expertise. Think of them as saved playbooks — you define the instructions once, and Claude follows them every time you invoke the skill.

How Skills Work

A skill is a Markdown file stored in .claude/skills/. When you type a slash command, Claude loads the skill file and follows its instructions for that task.

# Invoke the find-bug skill /find-bug src/auth/login.js # Invoke the test-design skill /test-design LoginController # Analyze a Jira requirement /analyze-requirement TICKET-123
Note: Skills are loaded fresh each time — edits to a skill file take effect immediately on the next invocation.
